## 🎯 Описание

Мощная и удобная система мониторинга ПК с современным графическим интерфейсом, разработанная на Python. Приложение предоставляет полную инвентаризацию системы, мониторинг производительности в реальном времени, отслеживание процессов и сетевых подключений, а также уникальный оверлей (HUD) для геймеров и профессионалов.

## ✨ Возможности

### 🖥️ **Полная системная инвентаризация**
- ✅ Детальная информация об операционной системе
- ✅ Характеристики процессора (модель, ядра, частота)
- ✅ Анализ оперативной памяти (объем, использование, модули)
- ✅ Информация о дисках и накопителях
- ✅ Данные о материнской плате и BIOS
- ✅ Информация о видеокартах и драйверах

### 📈 **Мониторинг в реальном времени**
- 🎮 **Интерактивный HUD** - прозрачный оверлей поверх всех окон
- 📊 **Живые графики** CPU, RAM и сетевой активности
- 🔥 **Мониторинг температуры** процессора
- 🔋 **Отслеживание заряда батареи** для ноутбуков
- ⚡ **Счетчик FPS** для оптимизации производительности

### 🧩 **Управление процессами**
- 📋 Список активных процессов с сортировкой по CPU
- 🔍 Поиск и анализ ресурсоемких приложений
- 👤 Информация о пользователях процессов
- 💾 Детальная статистика использования памяти

### 🌐 **Сетевой мониторинг**
- 📡 Состояние всех сетевых интерфейсов
- 🔗 Активные TCP/UDP подключения
- 📊 Статистика сетевого трафика
- 🚨 Мониторинг ошибок подключений

### 💾 **Экспорт данных**
- 📄 Сохранение отчетов в форматах TXT/JSON
- 🗂️ Автоматическое включение всех разделов
- 📅 Штамп времени генерации отчета

## 🚀 Установка

### Требования
- **Python 3.8+**
- **Windows** (основная платформа)
- **Права администратора** для полного доступа к системной информации

### Быстрая установка

```bash
# 1. Клонирование репозитория
git clone https://github.com/i000993i/system-monitoring-dbs
cd pc-monitoring-system

# 2. Установка зависимостей
pip install -r requirements.txt

# 3. Запуск приложения
python stats.py
```

### Альтернативный способ (готовый exe)

```bash
# Сборка исполняемого файла
pyinstaller --onefile --windowed --name="PC_Monitor" stats.py

# Запуск готового приложения
dist/PC_Monitor.exe
```

## 🎮 Использование

### Основной интерфейс
Приложение открывается с главным окном, разделенным на вкладки:

1. **🖥️ Инфо** - Полная системная информация
2. **🧩 Процессы** - Активные процессы и их ресурсы  
3. **🌐 Сеть** - Сетевые подключения и статистика
4. **📈 Мониторинг** - Живые графики производительности

### Горячие клавиши
- **`F8`** - Показать/скрыть оверлей (HUD)
- **`🔄 Обновить`** - Принудительное обновление всех данных
- **`💾 Отчёт`** - Сохранение полного отчета в файл

### Оверлей (HUD)
Прозрачное окно с ключевыми метриками:
- **FPS** - Частота кадров
- **CPU** - Загрузка процессора
- **RAM** - Использование памяти
- **Temperature** - Температура процессора
- **Battery** - Заряд батареи (для ноутбуков)

*Перетаскивайте оверлей мышью для изменения позиции*

## 🔧 Технологии

### Основной стек
- **🐍 Python 3.8+** - Основной язык программирования
- **🖼️ Tkinter** - Графический интерфейс (встроенный в Python)
- **📊 psutil** - Кроссплатформенная библиотека для системной информации

### Дополнительные библиотеки
- **🪟 WMI** - Windows Management Instrumentation (только для Windows)
- **🔨 PyInstaller** - Создание исполняемых файлов
- **⏰ datetime** - Работа с временем и датами
- **🔍 json** - Обработка конфигурационных файлов

### Системная информация
- **CPU**: Модель, частота, количество ядер/потоков, загрузка
- **RAM**: Объем, использование, детали модулей памяти
- **Диски**: Разделы, свободное место, файловые системы
- **Сеть**: Интерфейсы, подключения, статистика трафика
- **Процессы**: Список, CPU/RAM использование, пользователи
- **Температура**: Датчики CPU (через psutil)
- **Батарея**: Уровень заряда (для ноутбуков)
